# Hotel Menu Management System (Spring Boot)

This is a backend REST API project built using Spring Boot.

The project is designed to manage hotel menu items and demonstrates how backend systems work using REST APIs.

## 🚀 Features
- Basic GET APIs for menu
- Fetch list of menu items
- Simple REST API implementation

## 🛠️ Tech Stack
- Java
- Spring Boot
- REST APIs

## 📌 Project Status
This project is currently in progress. More features like POST, PUT, DELETE APIs and database integration will be added soon.

## 📂 How to Run
1. Clone the repository
2. Open in IDE (Eclipse/IntelliJ)
3. Run the main class
4. Access APIs on: http://localhost:8080

## 🎯 Learning Outcome
- Understanding of REST API development
- Hands-on experience with Spring Boot
- Basics of backend development
